Detailed Description

Singleton of the asynchronous port 1.

This class is the second level driver of the serial port 1 in asynchronous mode. It uses the CQueue framework of the FreeRTOS and FreeRTOS C++ wrapper as Rx/Tx buffers. It uses the CMutex framework of the FreeRTOS and FreeRTOS C++ wrapper to ensures that the port will not be used by more than one task at the time.
